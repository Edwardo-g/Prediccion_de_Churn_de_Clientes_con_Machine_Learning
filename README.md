# Predicción de Churn de Clientes con Machine Learning

| Sección                           | Contenido                                                                                                                                                                                                                                                                                                                   |
| --------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 📌 **Descripción General**        | Este proyecto desarrolla un modelo de Machine Learning para predecir la cancelación de clientes (churn) en una empresa de telecomunicaciones, con el objetivo de anticipar el abandono y apoyar estrategias de retención.                                                                                                   |
| 🎯 **Objetivo de Negocio**        | Identificar clientes con alta probabilidad de cancelar el servicio para permitir acciones preventivas que reduzcan la pérdida de clientes.                                                                                                                                                                                  |
| 🗂️ **Dataset**                   | Dataset con información de clientes, incluyendo variables demográficas, servicios contratados, tipo de contrato, método de pago, cargos y antigüedad.                                                                                                                                                                       |
| 🧹 **Preparación de Datos**       | - Limpieza y tratamiento de datos <br> - Transformación de variables binarias (0/1) <br> - Codificación de variables categóricas (OneHotEncoder) <br> - Escalado de variables (MinMaxScaler) <br> - Selección de variables relevantes                                                                                       |
| 🤖 **Modelado**                   | Se entrenaron y compararon múltiples modelos de clasificación: <br><br> - Dummy Classifier (baseline) <br> - Random Forest <br> - K-Nearest Neighbors (KNN)                                                                                                                                                                 |
| 📊 **Evaluación**                 | Se utilizaron métricas de desempeño: <br><br> - Accuracy <br> - Precision <br> - Recall <br> - F1-score <br> - Matriz de confusión                                                                                                                                                                                          |
| 🔎 **Principales Hallazgos**      | - El contrato **mes a mes** aumenta la probabilidad de churn <br> - La **antigüedad del cliente** reduce el riesgo de cancelación <br> - Servicios como **soporte técnico** y **seguridad online** están asociados a mayor retención <br> - Se priorizó el **recall** para detectar la mayor cantidad de clientes en riesgo |
| 💡 **Recomendaciones de Negocio** | - Incentivar contratos de largo plazo <br> - Fortalecer servicios de valor agregado (soporte, seguridad) <br> - Implementar estrategias de retención temprana para clientes nuevos                                                                                                                                          |
| 🛠️ **Tecnologías**               | Python (Pandas, NumPy), Scikit-learn, Matplotlib, Seaborn, Statsmodels                                                                                                                                                                                                                                                      |
| ⚠️ **Nota**                       | Proyecto desarrollado como parte de formación en análisis de datos, enfocado en la aplicación de Machine Learning a un caso de negocio.                                                                                                                                                                                     |
